# 🌾 AgriSmart Punjab: AI-Powered Farmer Decision Support System

## 🚀 Hackathon Project - Empowering Punjab Farmers with Smart Technology

**AgriSmart Punjab** is an innovative AI-driven web application designed to revolutionize farming practices in Punjab, Pakistan. Built with Streamlit, this platform provides farmers with intelligent soil analysis, crop recommendations, and government scheme information to optimize agricultural productivity and sustainability.

## 🎯 Problem Statement

Punjab's agriculture sector faces challenges like:
- Inefficient soil management leading to suboptimal yields
- Limited access to real-time farming insights
- Lack of awareness about government agricultural schemes
- Language barriers in accessing modern farming technology

## ✨ Features

### 🌱 Smart Soil Analysis
- **NPK Analysis**: Input nitrogen, phosphorus, and potassium levels for personalized recommendations
- **pH Monitoring**: Soil acidity/basicity assessment with actionable advice
- **Environmental Factors**: Temperature and humidity integration for comprehensive analysis

### 📊 Intelligent Recommendations
- AI-powered crop suitability analysis
- Fertilizer optimization suggestions
- Pest and disease prevention alerts

### 📢 Government Integration
- Real-time government scheme notifications
- Subsidy information for tractors and equipment
- Kissan Card loan details
- Weather alerts and advisories

### 🌐 Multilingual Support
- Urdu language interface for local farmers
- English support for broader accessibility

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **Data Processing**: Pandas, NumPy
- **Visualization**: Matplotlib
- **AI/ML**: Scikit-learn (planned for future enhancements)
- **Deployment**: Streamlit Cloud

## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/agri-smart-punjab.git
   cd agri-smart-punjab
   ```

2. **Create virtual environment**
   ```bash
   python -m venv .venv
   source .venv/Scripts/activate  # On Windows
   # or
   source .venv/bin/activate     # On macOS/Linux
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application**
   ```bash
   streamlit run app.py
   ```

5. **Open your browser** and navigate to `http://localhost:8501`

## 📱 Usage

1. **Access the Dashboard**: Open the app in your browser
2. **Input Soil Parameters**: Enter NPK values, pH, temperature, and humidity
3. **Get Recommendations**: Click "Generate Recommendation" for personalized farming advice
4. **Check Government Updates**: View sidebar for latest schemes and alerts
